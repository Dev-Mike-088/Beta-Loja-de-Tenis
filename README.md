## Sistema de Compra de Tênis em Java

## Descrição

Projeto simples em Java desenvolvido para praticar:

- Entrada de dados com `Scanner`
- Estruturas condicionais (`if/else`)
- Estrutura de repetição (`do while`)
- Organização de menu interativo
- Atualização de variáveis durante execução

O sistema simula a compra de um tênis em uma loja, permitindo ao usuário visualizar saldo, consultar preço e realizar a compra.

---

## Funcionalidades

-  Visualizar saldo atual
-  Consultar valor do tênis
-  Realizar compra (com verificação de saldo)
-  Inserir mais dinheiro
-  Encerrar o programa

---

## Tecnologias Utilizadas

- Java
- Classe `Scanner` para entrada de dados
- Estruturas condicionais
- Estrutura de repetição `do while`

---

## Como Funciona

1. O usuário informa quanto dinheiro possui.
2. Informa o valor do tênis.
3. O sistema exibe um menu interativo.
4. O usuário pode navegar pelas opções até escolher sair.

Se o saldo for suficiente, a compra é realizada e o valor é atualizado.

---

## Objetivo do Projeto

Treinar lógica de programação e estrutura de menus interativos em Java, simulando uma situação real de compra com controle de saldo.
